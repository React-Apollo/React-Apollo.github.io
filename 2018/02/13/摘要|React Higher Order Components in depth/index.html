什么是高阶组件 HOC?

HOC是包装另一个组件的 React 组件

通常是一个函数,作为类工厂,

“wrap”部分:

  1. Props 代理: HOC负责操作传递给包装组件W 的 props.
  2. HOC 扩展了 W的功能

HOC可以做什么

  • 代码重用,逻辑抽象和 bootstrap 式的抽象(一般->特殊的样式定义)
  • 渲染劫持
  • State抽象和操作
  • Props 操作
